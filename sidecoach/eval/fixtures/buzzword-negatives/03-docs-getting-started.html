Getting Started

Install the client, connect to your database, and run your first query in under five minutes.

Install the client

Add the package to your project with your package manager of choice. The client has no runtime dependencies and works in Node.js 18 and later.

npm install @example/client

Connect to your database

Create a connection using the URL from your dashboard. Connections are pooled by default, so you can reuse the same client across requests without exhausting the connection limit.

import { connect } from "@example/client";
const db = connect(process.env.DATABASE_URL);

Powerful queries with SQL

Write plain SQL and get typed results back. The client parses your query at build time and infers the return type from the column list, so a typo in a column name is caught before you run the code. Parameters are passed separately to prevent injection.

const rows = await db.query("select id, email from users where id = $1", [42]);

Handle errors

Every query returns a result object. Network failures and constraint violations throw typed errors you can catch and inspect. Use a transaction when you need several statements to succeed or fail together.
