Email for developers

Send transactional and marketing email from your application with a single API. Built on open standards, priced by what you send.

Send your first email this morning

Install the SDK, add your API key, and send a message in three lines of code. The dashboard shows delivery, opens, and bounces in real time, and the logs keep ninety days of history you can search by recipient or subject.

First-class developer experience

Test mode

Test mode captures every message without delivering it, so you can run your suite against the real API in continuous integration without sending mail to anyone. Captured messages appear in the dashboard exactly as a live send would.

Modular webhooks

Subscribe to delivery, bounce, and complaint events. Each payload is signed so you can verify it came from us, and failed deliveries retry on an exponential schedule for up to a day.

Write using a delightful editor

Compose templates with a plain editor or import your own HTML. Variables are type-checked against the data you pass at send time, so a missing field is caught before the message goes out rather than rendering as a blank in someone's inbox.

Develop emails using React

Build templates as React components and render them to HTML at send time. Components are testable and reusable across transactional and marketing messages, and the same component renders identically in every major mail client.

Reach humans, not spam folders

We manage IP warmup, authentication records, and feedback loops so your mail lands in the inbox. Dedicated IPs are available on higher plans, and the deliverability dashboard flags domains that need an SPF or DKIM record.

Everything in your control

Suppression lists, rate limits, and sending domains are configured per environment. Roles let you give a teammate access to staging without touching production, and every change is recorded in an audit log.

Email reimagined

Start on the free tier with three thousand messages a month and upgrade when you outgrow it. There is no contract, billing is by the message, and you can export your data and leave at any time.
