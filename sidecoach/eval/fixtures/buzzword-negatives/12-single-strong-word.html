Seamless integrations for your data stack

Connect 200 sources to your warehouse and keep them in sync. Define a model once and reuse it across every destination.

Two hundred connectors

We maintain connectors for the databases, SaaS tools, and event streams that teams use most. Each connector handles schema changes, incremental syncs, and rate limits so you do not have to write that logic yourself. New connectors ship every week.

Sync on your schedule

Run a sync every five minutes, every hour, or on demand from the API. Incremental syncs read only the rows that changed since the last run, which keeps warehouse costs down and avoids re-scanning large tables.

Models you can trust

Define transformations in SQL and version them with your code. Tests run before a model is published, so a broken join is caught in continuous integration rather than in a dashboard the next morning.

Monitoring and alerts

Every sync reports row counts, duration, and errors. Set an alert when a sync fails or when volume drops below a threshold, and route it to Slack, email, or a webhook. The run history is retained for ninety days.
