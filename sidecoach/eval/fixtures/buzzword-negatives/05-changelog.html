Changelog

Notable changes to the platform, newest first.

Version 2.4.0

Released May 6, 2025

  • Added CSV export to the reports page. Exports include all columns visible in the current view and respect active filters.
  • The dashboard now remembers the last selected date range across sessions.
  • Reduced cold-start latency for the serverless functions by about 200 milliseconds.

Version 2.3.2

Released April 22, 2025

  • Fixed a bug where pagination skipped the final row when the total count was an exact multiple of the page size.
  • Fixed an issue where the webhook retry schedule used the wrong backoff interval.
  • Updated the Postgres driver to 8.11 to address a connection-leak warning.

Version 2.3.1

Released April 9, 2025

  • Corrected a typo in the invoice email template.
  • The CLI now prints a clearer message when the configuration file is missing.

Version 2.3.0

Released March 30, 2025

  • Introduced team roles with read-only and admin permissions.
  • Added support for SAML single sign-on on the business plan.
  • Migrated the search index to reduce query latency on large workspaces.
