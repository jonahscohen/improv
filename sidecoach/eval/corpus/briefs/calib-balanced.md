# SaaS feature section (calibration brief)

- register: marketing
- audience: a prospective buyer scanning a product landing page deciding whether to start a trial.
- goal: communicate one core feature and prompt a trial signup.
- required content: a section heading, a short value paragraph, three supporting points, one representative UI/illustration slot, and a primary call-to-action.
- constraints: fits a standard 1200px content column + responsive down to mobile; one accent color on a neutral ground; existing sans-serif brand font.
- success criteria: the value is clear in one scan; the three points are scannable; the CTA is obvious; the section feels balanced and trustworthy.
