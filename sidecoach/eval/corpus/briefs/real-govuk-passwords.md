# Ask users for
Passwords (real brief, GOV.UK Design System pattern)

source: https://design-system.service.gov.uk/patterns/passwords/

Ask users for
Passwords
Help users to create and enter secure and memorable passwords.
When to use this pattern
You should follow this pattern whenever you need users to create or enter a password. Before using this pattern, you should also read the guidance on the User accounts pattern.
For technical considerations, you may also want to read the guidance for the Password input component.
